AL-Lightbox Plugin

The ultimate lightbox plugin. It's lightweight and simple to use.
Doesn't need jQuery.

Demo

Installation

Move the /al-lightbox folder into your project.

Include CSS

Add the scripts before the closing body tag.

<link rel="stylesheet" href="al-lightbox/dist/css/vendor.min.css">

Include JS

Add the css links in head tag.

<script src="al-lightbox/dist/js/vendor.min.js"></script>

Set HTML

All you need to set is the data-image attribute with the desired image's url. The defined image will be set as background of the element containing the attribute.

<div class="items">
   <div class="item gallery-photo" data-image="images/1.jpg"></div>
   <div class="item gallery-photo" data-image="images/2.jpg"></div>
   <div class="item gallery-photo" data-image="images/3.jpg"></div>

Usage

Init

Initilize plugin

Parameters

Name Type Description
selector string Element(s) selector to initalize the plugin

Example

alLightbox.init(selector);

Destroy

Destroys plugin (resets element to its original state)

Parameters

Name Type Description
selector string Element(s) selector to destroy the plugin

Example

alLightbox.destroy(selector);
